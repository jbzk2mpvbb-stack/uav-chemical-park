# UAV Swarm System for Chemical Industrial Park

## Full-Cycle Intelligent Control of Hazardous Material Transport

A multi-modal UAV swarm monitoring system designed for chemical industrial parks, transforming hazardous material transport from rule-based safety processes into a predictive, data-driven governance framework.

### Interactive Demos

| Demo | Description |
|------|-------------|
| **Control Center** | Park-wide monitoring dashboard with aerial map, multi-zone signal monitoring, 3-level risk indicators, and AI monitoring agent |
| **AR Operation Guide** | AR-assisted unloading procedure with 6-step guidance, real-time safety monitoring, and leak alert simulation |
| **UAV Detection** | Object detection + safety state classification from drone perspective, with normal/anomaly mode switching |
| **Realtime Monitor** | Simulated UAV monitoring with Sense → Judge → Act event feed and dynamic detection boxes |
| **Anomaly Dashboard** | Multi-signal anomaly detection (Pressure, Temperature, Flow, Gas) with AI analysis and system logging |

### Tech Stack

- React 18
- Real-time data visualization (SVG)
- Multi-modal sensor simulation
- Rule-based AI agent
- Responsive HUD/AR interface design

### Run Locally

```bash
npm install
npm start
```
